# Web Systems - Bautista Portfolio

This repository contains course projects for Web Systems (CCS 9318 AY225) by Cyrus Bautista.

## Folder Structure

### 📁 BAUTISTA-PRELIMS
**JPCS Website - Junior Philippine Computer Society**
- A complete prelim project featuring the official website for the Junior Philippines Computer Society
- Includes pages for About, Events, Membership, Officers, and more
- Built with HTML, CSS, and JavaScript with modern UI/UX design

**Files:**
- `index.html` - Main homepage
- `about.html` - Organization information
- `events.html` - Upcoming events
- `membership.html` - Membership details
- `officers.html` - Leadership information
- `main/assets/` - CSS, JavaScript, and image resources
- `README.md` - Project documentation

### 📁 BAUTISTA-MIDTERMLAB
**Metacritic Game Scraper**
- A web scraping project that extracts game data from Metacritic
- Built with Python Flask backend and vanilla JavaScript frontend
- Features real-time search, CSV export, and JSON API

**Files:**
- `app.py` - Flask web server with API endpoints
- `scraper.py` - Web scraping logic using BeautifulSoup
- `requirements.txt` - Python dependencies
- `templates/index.html` - Web interface template
- `static/css/style.css` - Styling with black/white design system
- `static/js/script.js` - Client-side functionality
- `README.md` - Project documentation

## Technologies Used

### BAUTISTA-PRELIMS
- HTML5
- CSS3
- JavaScript (Vanilla)

### BAUTISTA-MIDTERMLAB
- Python 3
- Flask
- BeautifulSoup4
- Requests
- Vanilla JavaScript

## How to Use

### Running the Metacritic Game Scraper
```bash
cd BAUTISTA-MIDTERMLAB
pip install -r requirements.txt
python app.py
```
Then open `http://localhost:5000` in your browser.
